2. Update

******************************************************************************

1.Installation
==============

Zur Installation sind folgende Schritte durchzuführen:

1.1 Im Ordner adm_plugins einen neuen Ordner mit dem Namen kategoriereport erstellen

1.2 Alle Dateien (und Ordner) des Ordners kategoriereport aus der entpackten Datei kategoriereport_2.0.0.zip in diesen Ordner kopieren.

1.3 In der Datei my_body_bottom.php (Ordner adm_themes/modern) folgende Zeilen einfügen:

  1.3.1  Erste Möglichkeit
      Darstellung des Links zum Plugin im vorhandenen Module-Menü

      - dazu VOR der vorhandenen Anweisung "$moduleMenu->show();" diese Zeile einfügen:
         include(SERVER_PATH.'/adm_plugins/kategoriereport/kategoriereport.php');

   1.3.2  Zweite Möglichkeit
      Darstellung des Links zum Plugin in einem eigenen Plugin-Menü

       - dazu diese Zeilen einfügen (falls noch nicht vorhanden):
         $pluginMenu = new Menu('plugins', 'Plugins');
         $pluginMenu->show(); 

      - und innerhalb dieser Anweisungen den bereits bekannten include-Befehl einfügen
         $pluginMenu = new Menu('plugins', 'Plugins');
         include(SERVER_PATH. '/adm_plugins/kategoriereport/kategoriereport.php');
         $pluginMenu->show(); 
            
   Hinweis: Die include-Anweisung sollte nicht innerhalb einer if-Abfrage stehen. 
            Die Berechtigung, wer das Plugin aufrufen darf, wird über das Einstellungen-Modul festgelegt.
            Standardeinstellung ist: Rolle Webmaster und Rolle Mitglied

1.4 Admidio starten und das Plugin Kategoriereport ausführen

2. Update
=========

2.1 Alle Dateien im Pluginverzeichnis von Kategoriereport löschen.

2.2 wie unter Punkt 1.2 beschrieben

2.3 wie unter Punkt 1.3 beschrieben

2.4 * * * HINWEIS   * * *
    Mit dieser Version werden einige Konfigurationsdaten in einem geänderten Format in der Datenbank gespeichert. 
    Es ist leider nicht möglich alle vorhandenen Konfigurationsdaten in das neue Format zu überführen.
    Zum Bereinigen der Datenbank sollten daher alle vorhandenen,alten Konfigurationsdaten in der Admidio-Datenbank gelöscht werden.

    Dazu gibt es mehrere Möglichkeiten:

    2.4.1 Entweder nach dem Start des Plugins
          - über Einstellungen-Optionen die Deinstallationsroutine aufrufen

    2.4.2 oder vor dem Start des Plugins
          - folgendes PHP-Programm über die Adresszeile des Browsers ausführen: 
            <Pfad der Admidio-Installation>/adm_plugins/<Name des Pluginordners von Kategoriereport>/preferences_function.php?mode=2
            z.B. www.admidio.meinewebseite.de/adm_plugins/kategoriereport/preferences_function.php?mode=2

    2.4.3 oder vor dem Start des Plugins
           - Über phpMyAdmin in der Admidio-Datenbank die Tabelle adm_plugin_preferences suchen.
             In dieser Tabelle alle Einträge, die mit PKR_ beginnen, löschen

2.5 Admidio starten und das Plugin Kategoriereport ausführen
